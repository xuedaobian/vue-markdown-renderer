贡献指南 ​

欢迎贡献！

常用开发命令：

  • pnpm dev / pnpm play — 启动 playground
  • pnpm build — 构建库
  • pnpm test — 运行测试
  • pnpm docs:dev — 启动文档站点

文档贡献：

  1. 新页面写入 docs/ 或 docs/zh/
  2. 更新 docs/.vitepress/config.ts 的侧边栏
  3. 运行 pnpm docs:dev 预览
  4. 提交 PR 并等待审阅
